# ANPR-camera's in België — een tijdlijn

Een compacte, statische tijdlijn van publieke uitspraken en gedocumenteerde
toepassingen van ANPR-camera's. Ieder moment bevat zijn directe bron. De site
heeft geen buildstap, JavaScript of externe afhankelijkheden.

## Lokaal bekijken

```sh
python3 -m http.server 4173
```

Open daarna <http://localhost:4173>.

De inhoud en bronverwijzingen zijn bijgewerkt tot en met 11 juli 2026.
